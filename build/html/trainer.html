Trainer¶

class src.trainer.Trainer(config: configs.main.config_template.DefaultConfig, **kwargs)[source]¶

Customize every aspect of training via flags

Parameters

  • max_epochs¶ (int) – Number of epochs in training

  • default_root_dir¶ (str) – Path to root directory of model experiments

  • checkpoint_callback¶ (bool) – Parameter responsible for saving model

  • callbacks¶ (list) – List of callbacks used with training

  • logger¶ (bool) – Flag responsible for default progress bar

  • gpus¶ – Which GPUs to train on.

  • profiler¶ – To profile individual steps during training and assist in identifying bottlenecks.

  • weigths_summary¶ (str) –

    Prints a summary of the weights when training begins. Supported options:

    • top: only the top-level modules will be recorded

    • full: summarizes all layers and their submodules
