class src.transformations.flip.Flip(dims: List[int], **kwargs)[source]¶

Bases: src.base.transformations.BaseTransformation

flip(data, dims) -> Tensor

Reverse the order of a n-D tensor along given axis in dims.

Parameters

  • data¶ (Tensor) – the data tensor.

  • dims¶ (a list or tuple) – axis to flip on

Example:

>>> x = torch.arange(8).view(2, 2, 2)
>>> x
tensor([[[ 0,  1],
         [ 2,  3]],

        [[ 4,  5],
         [ 6,  7]]])
>>> torch.flip(x, [0, 1])
tensor([[[ 6,  7],
         [ 4,  5]],

        [[ 2,  3],
         [ 0,  1]]])

argmax(data) -> LongTensor

Returns the indices of the maximum value of all elements in the data tensor.

This is the second value returned by torch.max(). See its documentation for the exact semantics of this method.

Parameters

data¶ (Tensor) – the data tensor.

Example:

>>> a = torch.randn(4, 4)
>>> a
tensor([[ 1.3398,  0.2663, -0.2686,  0.2450],
        [-0.7401, -0.8805, -0.3402, -1.1936],
        [ 0.4907, -1.3948, -1.0691, -0.3132],
        [-1.6092,  0.5419, -0.2993,  0.3195]])
>>> torch.argmax(a)
tensor(0)
argmax(data, dim, keepdim=False) → LongTensor¶

Returns the indices of the maximum values of a tensor across a dimension.

This is the second value returned by torch.max(). See its documentation for the exact semantics of this method.

Parameters

  • data¶ (Tensor) – the data tensor.

  • dim¶ (int) – the dimension to reduce. If None, the argmax of the flattened data is returned.

  • keepdim¶ (bool) – whether the output tensor has dim retained or not. Ignored if dim=None.

Example:

>>> a = torch.randn(4, 4)
>>> a
tensor([[ 1.3398,  0.2663, -0.2686,  0.2450],
        [-0.7401, -0.8805, -0.3402, -1.1936],
        [ 0.4907, -1.3948, -1.0691, -0.3132],
        [-1.6092,  0.5419, -0.2993,  0.3195]])
>>> torch.argmax(a, dim=1)
tensor([ 0,  2,  0,  1])
